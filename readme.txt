== Description ==

Manage WooCommerce discounts and apply a dynamic discount depending on the total number of cart items.

Discounts will be visible in cart and checkout page.

In order to enable the “Dynamic Discounts”, go to WooCommerce -> Dynamic Discounts.

== Installation ==

= Automatic Installation =
Go to WordPress dashboard, click  Plugins / Add New  , type 'WooCommerce Dynamic Discounts' and hit Enter.
Install and activate plugin, visit WooCommerce > Dynamic Discounts.
